# Gatsby Contentful Starter

Easy static site with dynamic content.

## getting started

```sh
npm install

# create contentful space
mv .example.env .env
# add secrets to .env

npm run dev
```

## fresh deploy

- initialize new git repo
- upload to github
- on netlify, import site from GH
- on netlify, set env vars (`Deploy settings > Environment`)
- push to deploy
